Core API

CreateWindow

Creates the main window container. Must be called before creating any tabs or components.

Argument Type Description
AutoExec boolean Re-executes script on teleport via queue_on_teleport.
AutoExecUrl string URL of the raw script to download and execute for AutoExec.
AutoSave boolean Auto-saves component state changes.
ConfigName string Filename key used for config persistence.
DisplayOrder number Render layering display order index of the ScreenGui.
Icon string Lucide icon name shown in the corner.
Name string Name property of the ScreenGui instance.
Position UDim2 Initial screen position coordinates of the window.
Size UDim2 Window dimensions.
Subtitle string Subtitle text below the title.
Title string Window title displayed in the top bar.

Code Snippet

Lua 
local window = MonoUI.CreateWindow({
    Title      = "mono ui",
    Subtitle   = "premium modular library",
    Size       = UDim2.fromOffset(600, 400),
    Icon       = "shield",
    ConfigName = "my_config",
    AutoSave   = true,
    AutoExec   = true,
})
