CreateSlider

A draggable range input with min/max bounds and optional decimal precision.

Argument Type Description
callback function Fires with the current numeric value.
default number Initial value.
flag string Unique key for config saving.
max number Maximum value of the range.
min number Minimum value of the range.
text string Label for the slider row.

Code Snippet

Lua 
tab:CreateSlider({
    text     = "FOV Range",
    min      = 10,
    max      = 200,
    default  = 90,
    flag     = "aimbot_fov",
    callback = function(value)
        print("FOV:", value)
    end,
})

Complete Script Usage

Lua (Full Setup) 
-- Load the MonoUI Library
local MonoUI = loadstring(game:HttpGet("https://github.com/BloodLetters/mono-ui/releases/latest/download/Release.luau"))()

-- Create the main Window
local window = MonoUI.CreateWindow({
    Title      = "mono ui",
    ConfigName = "mono_config",
    AutoSave   = true,
})

-- Create a Tab
local tab = window:CreateTab({ text = "Main", icon = "home" })

-- Create the component
tab:CreateSlider({
    text     = "FOV Range",
    min      = 10,
    max      = 200,
    default  = 90,
    flag     = "aimbot_fov",
    callback = function(value)
        print("FOV:", value)
    end,
})
