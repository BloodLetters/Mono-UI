🌌 MonoUI Library Documentation

Welcome to the official developer documentation for MonoUI. MonoUI is a state-of-the-art, high-performance, dark-mode Roblox UI library focusing on beautiful aesthetics, modular expansion, auto-saving configurations, and live search filtering.

Tip: All interactive components automatically sync their values with the global configuration database when modified.

🚀 Getting Started

To initialize MonoUI in your script executor, paste and run the following loadstring bootstrapper: 

local MonoUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/mono-ui/main/dist/mono-ui.luau"))()

✨ Key Features

Global Search Filter

A built-in Live Search Bar inside the TopBar filters components instantly as you type.

Draggable Control HUD

Rounded HUD carrying setting toggles. Supports safe drag-threshold triggers.

AutoExec Teleportation

Automatically queues execution on server hops using queue_on_teleport.

Automatic Save & Load

Auto-serializes element configurations, including Color3 and KeyCodes.

🪟 CreateWindow

Creates the core glassmorphic container instance for your library controls.

local window = MonoUI.CreateWindow({
    Title = "mono ui",
    Subtitle = "premium modular library",
    Size = UDim2.fromOffset(600, 400),
    Icon = "shield",
    ConfigName = "mono_demo_config", -- Saved as mono_demo_config.json
    AutoSave = true,                 -- Saves component changes instantly
    AutoExec = true                  -- Automatically reload script on teleport
})

🔔 Notify Toast

Spawns a translucent toast notification card in the bottom-right corner of the screen.

MonoUI.Notify({
    title = "System Alert",
    content = "MonoUI has loaded successfully.",
    icon = "check-circle",
    duration = 4
})

🖥️ SetWatermark

Displays a floating utility overlay tracking execution stats, ping, and local system clock.

MonoUI.SetWatermark({
    visible = true,
    text = "MonoUI Premium v1.0"
})

🎛️ CreateControlHUD

Creates a small draggable rounded quick settings menu carrying icons only.

MonoUI.CreateControlHUD({
    {
        icon = "swords",
        default = false,
        callback = function(active)
            print("Silent Aim: " .. tostring(active))
        end
    },
    {
        icon = "eye",
        default = true,
        callback = function(active)
            print("ESP: " .. tostring(active))
        end
    }
})

📂 CreateTab

Appends a navigation tab item to the window sidebar.

local combatTab = window:CreateTab({
    text = "Combat",
    icon = "swords"
})

🗂️ CreateSection

Groups consecutive components together under a section banner.

combatTab:CreateSection({
    text = "Combat Hacks"
})

🔘 Toggle Button

A simple on/off switch component.

tab1:CreateToggle({
    text = "Silent Aim",
    default = false,
    flag = "silent_aim_state",
    callback = function(state)
        print("Silent Aim: " .. (state and "ENABLED" or "DISABLED"))
    end
})

🎚️ Numeric Slider

A draggable range input slider with support for decimals and unit suffixes.

tab1:CreateSlider({
    text = "FOV Range",
    min = 10,
    max = 200,
    default = 90,
    decimals = 1,              -- Float value spacing
    suffix = "°",              -- Suffix character
    flag = "aimbot_fov",
    callback = function(value)
        print("FOV Range set to: " .. value)
    end
})

🔽 Dropdown Select

A collapsed scrollable select list supporting single or multiple selections.

tab1:CreateDropdown({
    text = "Target Mode",
    list = {"Distance", "FOV", "Health"},
    default = "Distance",
    multiple = false,
    flag = "target_mode",
    callback = function(selection)
        print("Selected: " .. tostring(selection[1] or selection))
    end
})

🔤 Text Input

A focusable text box allowing players to type strings.

tab1:CreateInput({
    text = "Aimbot Key",
    placeholder = "Type text...",
    default = "Headshot",
    flag = "aimbot_key_text",
    callback = function(value)
        print("Aimbot text value set to: " .. value)
    end
})

🖱️ Button Clicker

Standard clickable trigger button.

tab1:CreateButton({
    text = "Reset Settings",
    callback = function()
        print("Settings reset successfully.")
    end
})

🎨 Color Picker

Provides a visual color selector pop-up canvas.

tab1:CreateColorPicker({
    text = "ESP Color",
    default = Color3.fromRGB(0, 162, 255),
    flag = "esp_theme_color",
    callback = function(color)
        print("ESP Theme Color updated: ", color)
    end
})

⌨️ Keybind Capturer

Allows users to register custom keyboard binds. Supports pressing Escape to clear.

tab1:CreateKeybind({
    text = "Menu Toggle Key",
    default = Enum.KeyCode.RightControl,
    flag = "menu_toggle_key",
    callback = function(key)
        print("Pressed hotkey: " .. key.Name)
    end
})

🎯 Skeletal Target Selection

Advanced interactive R6 body layout selection. Limbs can be toggled or disabled.

tab1:CreateTargetBody({
    text = "Target Limbs",
    default = {"Head", "Torso"},
    disabledParts = {"RightArm", "LeftArm"},
    flag = "body_parts",
    callback = function(selectedParts)
        print("Active parts: " .. table.concat(selectedParts, ", "))
    end
})

👥 Player List & ESP

Generates a list of active players in the server. Clicking TP teleports your character, while ESP turns on highlights.

tab1:CreatePlayerList({
    text = "Active Server Players",
    height = 280
})

🖥️ In-App Console

Creates an in-app console widget showing activity logs.

local logger = tab1:CreateLogger({
    text = "Console Output Logs",
    height = 280
})

-- Logger syntax:
logger:Log("INFO", "Starting engine...")
logger:Log("SUCCESS", "ESP fully loaded.")
logger:Log("WARNING", "Server lag detected.")
logger:Log("ERROR", "Access denied.")

💻 Complete Script Example

This is a complete script using all the custom methods in the library:

local MonoUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/mono-ui/main/dist/mono-ui.luau"))()

-- Configure stats overlay
MonoUI.SetWatermark({
    visible = true,
    text = "MonoUI Premium",
})

-- Create main window
local window = MonoUI.CreateWindow({
    Title = "mono ui",
    Subtitle = "premium modular library",
    Size = UDim2.fromOffset(600, 400),
    Icon = "shield",
    ConfigName = "mono_demo_config",
    AutoSave = true,
    AutoExec = true,
})

-- Setup console tab
local consoleTab = window:CreateTab({
    text = "Console",
    icon = "terminal",
})
consoleTab:CreateSection({
    text = "System Action Logs"
})
local logger = consoleTab:CreateLogger({
    text = "Mono Console Output",
    height = 280,
})

-- Setup combat tab
local combatTab = window:CreateTab({
    text = "Combat",
    icon = "swords",
})
combatTab:CreateSection({
    text = "Combat Hacks"
})
combatTab:CreateToggle({
    text = "Silent Aim",
    default = false,
    flag = "silent_aim",
    callback = function(state)
        logger:Log("INFO", "Silent Aim toggled: " .. tostring(state))
    end
})

-- Setup floating Control HUD
MonoUI.CreateControlHUD({
    {
        icon = "swords",
        default = false,
        callback = function(active)
            logger:Log("INFO", "HUD - Silent Aim: " .. tostring(active))
        end
    },
    {
        icon = "eye",
        default = true,
        callback = function(active)
            logger:Log("INFO", "HUD - ESP: " .. tostring(active))
        end
    }
})
