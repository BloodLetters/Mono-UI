Overview

MonoUI Documentation

A premium, modular Roblox UI library with dark-mode aesthetics, auto-saving config, live search, and autoexec support.

All interactive components automatically sync their values to the global config when modified. Use flag to identify each element.
Getting Started › Quick Start
Setup

Quick Start

Paste the loadstring bootstrapper into your script to initialize MonoUI.

Lua 
local MonoUI = loadstring(game:HttpGet("https://github.com/BloodLetters/mono-ui/releases/latest/download/Release.luau"))()

Overview

Features

MonoUI ships with a full suite of UI primitives and productivity features out of the box.

Live Search

Built-in search bar filters components in real-time as you type.

Auto Save & Load

Serializes all component states including Color3 and KeyCodes.

AutoExec

Queues script re-execution on server teleport automatically.

Control HUD

Draggable floating quick-toggle HUD with icon-only buttons.

Core › CreateWindow
Core API

CreateWindow

Creates the main window container. Must be called before creating any tabs or components.

Argument Type Description
Title string Window title displayed in the top bar.
Subtitle string Subtitle text below the title.
Size UDim2 Window dimensions.
Icon string Lucide icon name shown in the corner.
ConfigName string Filename key used for config persistence.
AutoSave boolean Auto-saves component state changes.
AutoExec boolean Re-executes script on teleport via queue_on_teleport.
Lua 
local window = MonoUI.CreateWindow({
    Title      = "mono ui",
    Subtitle   = "premium modular library",
    Size       = UDim2.fromOffset(600, 400),
    Icon       = "shield",
    ConfigName = "my_config",
    AutoSave   = true,
    AutoExec   = true,
})
Core › Notify
Core API

Notify

Spawns a toast notification card in the bottom-right corner. Multiple calls stack automatically.

Argument Type Description
title string Bold heading text of the notification.
content string Body message shown below the title.
icon string Lucide icon name rendered on the left.
duration number Seconds before auto-dismiss (default: 4).
Lua 
MonoUI.Notify({
    title    = "System Alert",
    content  = "MonoUI has loaded successfully.",
    icon     = "check-circle",
    duration = 4,
})

Core › CreateControlHUD
Core API

CreateControlHUD

Creates a small draggable icon-only quick-toggle HUD. Each entry is a toggle button.

Argument Type Description
icon string Lucide icon name for this HUD button.
default boolean Initial active state of the toggle.
callback function Called with active: boolean on toggle.
Lua 
MonoUI.CreateControlHUD({
    { icon = "swords", default = false, callback = function(v) print("Aim:", v) end },
    { icon = "eye",    default = true,  callback = function(v) print("ESP:", v) end },
})

Components › Toggle
Components

Toggle

A checkbox-style switch for boolean values. State is persisted to config when a flag is provided.

Argument Type Description
text string Label for the toggle.
default boolean Initial state (true/false).
flag string Unique key for config saving.
callback function Fires with the new boolean state.
Lua 
tab:CreateToggle({
    text     = "Silent Aim",
    default  = false,
    flag     = "silent_aim",
    callback = function(state)
        print("Silent Aim:", state)
    end,
})
Components › Slider
Components

Slider

A draggable range input with min/max bounds and optional decimal precision.

Argument Type Description
text string Label for the slider row.
min number Minimum value of the range.
max number Maximum value of the range.
default number Initial value.
flag string Unique key for config saving.
callback function Fires with the current numeric value.
Lua 
tab:CreateSlider({
    text     = "FOV Range",
    min      = 10,
    max      = 200,
    default  = 90,
    flag     = "aimbot_fov",
    callback = function(value)
        print("FOV:", value)
    end,
})
Components › Dropdown
Components

Dropdown

A collapsed select list supporting single or multiple selections from a list of options.

Argument Type Description
text string Label for the dropdown row.
list table Array of string options.
default string Pre-selected option.
multiple boolean Allow multi-select if true.
flag string Unique key for config saving.
callback function Fires with selected value(s) as a table.
Lua 
tab:CreateDropdown({
    text     = "Target Mode",
    list     = { "Distance", "FOV", "Health" },
    default  = "Distance",
    multiple = false,
    flag     = "target_mode",
    callback = function(selection)
        print("Selected:", selection[1])
    end,
})
Components › Input
Components

Input

A focusable text box allowing the user to type a string value.

Argument Type Description
text string Label displayed beside the input.
placeholder string Placeholder hint inside the text field.
default string Pre-filled value on load.
flag string Unique key for config saving.
callback function Fires with the submitted string on Enter.
Lua 
tab:CreateInput({
    text        = "Custom Tag",
    placeholder = "Enter text...",
    default     = "",
    flag        = "custom_tag",
    callback    = function(value)
        print("Tag set to:", value)
    end,
})

ColorPicker

A visual color selector pop-up with hue/saturation canvas. Fires in real-time as the user drags.

Argument Type Description
text string Label for the color picker row.
default Color3 Initial color value.
flag string Unique key for config saving.
callback function Fires with the new Color3 on change.
Lua

    default  = Color3.fromRGB(0, 162, 255),
    flag     = "esp_color",
    callback = function(color)

})
Components › Keybind
Components

Keybind

Records a keyboard shortcut from the user. Press Escape to clear the binding.

Argument Type Description
text string Label for the keybind row.
default KeyCode Default key binding.
flag string Unique key for config saving.
callback function Fires with the registered KeyCode when pressed.
Lua 
tab:CreateKeybind({
    text     = "Toggle Menu",
    default  = Enum.KeyCode.RightControl,
    flag     = "menu_key",
    callback = function(key)
        print("Key pressed:", key.Name)
    end,
})
Components › TargetBody
Components

TargetBody

An interactive R6 body-part selector. Individual limbs can be toggled or disabled entirely.

Argument Type Description
text string Section label.
default table Array of initially selected part names.
disabledParts table Array of part names that cannot be toggled.
flag string Unique key for config saving.
callback function Fires with an array of currently selected parts.
Lua 
tab:CreateTargetBody({
    text          = "Hitbox Parts",
    default       = { "Head", "Torso" },
    disabledParts = { "RightArm", "LeftArm" },
    flag          = "hitbox_parts",
    callback      = function(parts)
        print("Parts:", table.concat(parts, ", "))
    end,
})

Components › Logger
Components

Logger

An in-app console widget that displays color-coded activity logs. Supports INFO, SUCCESS, WARNING, and ERROR levels.

Argument Type Description
text string Console panel header label.
height number Pixel height of the log area.
Lua 
local log = tab:CreateLogger({ text = "Console", height = 280 })

log:Log("INFO",    "Script loaded.")
log:Log("SUCCESS", "ESP enabled.")
log:Log("WARNING", "High ping detected.")
log:Log("ERROR",   "Target not found.")

Full Script

A complete working example using all major MonoUI features.

Lua 
local MonoUI = loadstring(game:HttpGet("https://raw.githubusercontent.com/YOUR_USERNAME/mono-ui/main/dist/mono-ui.luau"))()

MonoUI.SetWatermark({ visible = true, text = "MonoUI Premium" })

local window = MonoUI.CreateWindow({
    Title      = "mono ui",
    Subtitle   = "premium modular library",
    Size       = UDim2.fromOffset(600, 400),
    Icon       = "shield",
    ConfigName = "my_config",
    AutoSave   = true,
    AutoExec   = true,
})

-- Console tab
local consoleTab = window:CreateTab({ text = "Console", icon = "terminal" })
consoleTab:CreateSection({ text = "System Logs" })
local logger = consoleTab:CreateLogger({ text = "Output", height = 280 })

-- Combat tab
local combatTab = window:CreateTab({ text = "Combat", icon = "swords" })
combatTab:CreateSection({ text = "Aimbot" })
combatTab:CreateToggle({
    text = "Silent Aim", default = false, flag = "silent_aim",
    callback = function(v) logger:Log("INFO", "Silent Aim: " .. tostring(v)) end,
})
combatTab:CreateSlider({
    text = "FOV", min = 10, max = 200, default = 90, flag = "fov",
    callback = function(v) logger:Log("INFO", "FOV: " .. v) end,
})

logger:Log("SUCCESS", "All tabs loaded.")
